 Mad Libs Game (C)

A simple and fun **Mad Libs word game** written in C.  
The program takes user input and generates a humorous story.

 Features
- User input using `fgets`
- String handling
- Dynamic story generation
- Beginner-friendly C project

 How to Run
1. Open in Dev-C++
2. Compile the program
3. Run and enter words when prompted

Concepts Used
- Strings in C
- `fgets` and formatted output
- Arrays and basic input handling
